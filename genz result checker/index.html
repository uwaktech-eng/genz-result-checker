What this portal offers

Everything is designed to be easy to deploy, easy to manage, and smooth to use on mobile and desktop.

🔐

Admin account system

Admins can sign up, log in, manage settings, save student records, and publish results securely.

🧾

Elegant report cards

Each student result appears in a polished report-card block with score, grade, remark, and teacher comment.

📱

Responsive design

Works well across Android phones, tablets, and desktop browsers with smooth transitions and button states.

🗂️

Google Sheets database

All records are stored in clearly organized tabs created automatically by the Apps Script backend.

🚀

Static hosting friendly

Deploy the frontend on GitHub Pages or Vercel without needing Node.js or a complex server setup.

🖨️

Print to PDF

Students can print their result report directly from the browser and save it as PDF when needed.
